# 踩地雷 Tsai Delay
楓夜景夏遇建成四校聯合暑訓成發作品 by [張雲淞（章魚）](https://instagram.com/slintar.otp)

## 遊戲簡介
### 規則說明
遊戲介面由9×9個方格組成，可分為地雷區與安全區，遊戲目的為尋找到所有安全的區域，並標記所有地雷的位置。

### 操作方法
- `左鍵單擊`：開啟方格或點擊按鈕
- `右鍵單擊`：標示或取消標示地雷位置

## 備註與謝誌
### 備註
本作品所有內容皆為自行撰寫，**並未使用任何AI等資源進行生成**。
